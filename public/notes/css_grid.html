Grid Intro

Grid Intro: Overview

• CSS Grid is the most powerful layout system in CSS

• First CSS tool to seriously address layout issues

• Supports two dimensions: rows and columns

• Like Flexbox: system of parents and children

Grid Intro: Why Learn

1. Easy (?) to create two dimensional layout

2. Simpler/reduced HTML markup

3. Browser support (support by all major browsers)

4. Skip frameworks like bootstrap

Grid Intro: Terminology

Grid Container:


  <div class="container"> // Grid Container
    <div class="item"></div> 
    <div class="item"></div> 
    <div class="item"></div>
  </div>

• Element which display: grid is applied

• Direct parent of all the grid items

Grid Intro: Terminology

Grid Item:


  <div class="container">
    <div class="item"></div> // Grid Item
    <div class="item"></div> // Grid Item
    <div class="item"></div> // Grid Item
  </div>

• Direct children of the grid container

Rows and Columns

Rows and Columns

• Open example-one.html

• Apply display:grid to .container

• Apply the following to .container:


    grid-template-columns: 100px 100px;
    grid-template-rows: 50px 50px 50px;
    grid-gap: 3px;
Rows and Columns

• Use auto to maximize grid item's sizing:


    grid-template-columns: 100px auto 100px;
    grid-template-rows: 50px 50px;
    grid-gap: 3px;

• Apply auto to each column for same-width layout:


    grid-template-columns: auto auto auto;

Fraction Units

Fraction Units

• Grid intros a new unit of measurement: fr

• Represents a fraction of available space in the grid

• In example-2.html, replace auto with fr

• Can we make the middle column larger?


    grid-template-columns: 1fr 2fr 1fr;

Grid Item Sizing

Grid Item Sizing

• So far, we have controlled grid items via the parent

• Grid provides grid item rules for advanced layout

• Open example-3 and change:


    grid-template-rows: 40px 200px 40px;

• What happens to the third row? (Check inspector!)

Grid Item Sizing

• Adjust the width of the header via grid-column:


  .header {
    grid-column-start: 1;
    grid-column-end: 3;
  }

• Start .header on 1st grid line and end on 3rd

• Add these rules to .footer

Grid Item Sizing

• Let's use 12 columns instead of 3 with repeat:


    grid-template-columns: repeat(12, 1fr);

• Update the grid-column properties for the items

• Hint: Try grid-column-end: -1;
